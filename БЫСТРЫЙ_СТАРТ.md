# 🤖 TradingAgents — Быстрый старт

## Запуск

Дважды кликните на ярлык **TradingAgents** на рабочем столе.

Или вручную:
```
launch.bat
```

---

## Настройка API-ключей

Откройте файл `.env` и укажите ключ нужного LLM-провайдера:

```env
# Выберите один или несколько:
OPENAI_API_KEY=sk-...          # OpenAI (GPT-5.x)
GOOGLE_API_KEY=AIza...         # Google (Gemini)
ANTHROPIC_API_KEY=sk-ant-...   # Anthropic (Claude)
XAI_API_KEY=...                # xAI (Grok)
DEEPSEEK_API_KEY=...           # DeepSeek

# Для рыночных данных (опционально, есть бесплатный tier):
ALPHA_VANTAGE_API_KEY=...      # alphavantage.co
```

> **Минимум нужен один LLM API-ключ.** Бесплатно можно попробовать с Google Gemini: https://aistudio.google.com/apikey

---

## Как работает CLI

После запуска вы увидите интерактивный интерфейс:

1. **Ticker** — введите тикер акции (например `AAPL`, `NVDA`, `TSLA`)
2. **Date** — дата анализа в формате `YYYY-MM-DD`
3. **LLM Provider** — выберите провайдера (OpenAI / Google / Anthropic и др.)
4. **Model** — выберите модель
5. **Research depth** — глубина анализа (1–3)

Агенты начнут анализ: фундаментальный → сентимент → новости → технический → дебаты → торговое решение.

## Структура проекта

```
TradingAgents/
├── launch.bat              ← скрипт запуска
├── .env                    ← ваши API-ключи (заполнить!)
├── cli/                    ← исходник CLI
├── tradingagents/          ← ядро фреймворка
│   ├── graph/              ← LangGraph граф агентов
│   ├── agents/             ← отдельные агенты
│   └── default_config.py   ← конфигурация по умолчанию
├── main.py                 ← пример запуска через Python
└── venv/                   ← виртуальное окружение (создано)
```

---

## Использование через Python

```python
from tradingagents.graph.trading_graph import TradingAgentsGraph
from tradingagents.default_config import DEFAULT_CONFIG

config = DEFAULT_CONFIG.copy()
config["llm_provider"] = "google"        # google, openai, anthropic, ...
config["deep_think_llm"] = "gemini-2.5-pro"
config["quick_think_llm"] = "gemini-2.5-flash"

ta = TradingAgentsGraph(debug=True, config=config)
_, decision = ta.propagate("NVDA", "2025-01-15")
print(decision)
```

## Checkpoint / Resume

```bat
REM Запуск с сохранением прогресса (возобновление при сбое):
venv\Scripts\python.exe -m cli.main --checkpoint

REM Сброс всех сохранённых состояний:
venv\Scripts\python.exe -m cli.main --clear-checkpoints
```
